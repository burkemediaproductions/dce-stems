DCE Stems single-page site starter

FILES TO EDIT:
- assets/logo-placeholder.svg -> replace with your real logo file and update references in index.html if filename changes
- assets/hero-video-placeholder.mp4 -> replace with your real background video
- index.html -> edit all text, swap video placeholders, and adjust CTA wording
- form in index.html -> already set for Netlify, but you can add/remove fields as needed

NETLIFY NOTES:
1. Push the contents of this folder to GitHub.
2. Connect the repo to Netlify.
3. Deploy normally as a static site.
4. Netlify will detect the form after deployment because the form markup is included in the HTML.
5. After the first deploy, submit a test form on the live site so it appears in Netlify Forms.

OPTIONAL VIDEO EMBEDS:
- Replace the placeholder blocks with YouTube, Vimeo, or local video embed code.
- If using iframe embeds, wrap them in the same container classes so styling remains intact.

LEGAL / CONTENT NOTE:
This starter uses careful educational wording and avoids strong medical claims. Keep final copy compliant with your provider and legal guidance.
